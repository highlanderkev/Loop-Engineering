# Loop Engineering

A Python project scaffold for Loop Engineering.

## Getting Started

### Prerequisites

- Python 3.10+

### Installation

Install the package in development mode:

```bash
pip install -e ".[dev]"
```

### Running Tests

```bash
pytest
```

### Linting

```bash
ruff check src tests
```
